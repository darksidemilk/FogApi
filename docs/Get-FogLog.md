# Get-FogLog

## SYNOPSIS
Get a auto updating fog log

## SYNTAX

```
Get-FogLog [-static] [<CommonParameters>]
```

## DESCRIPTION
Ust get-content -wait to show a dynamic fog log or use -static to just see the current contents

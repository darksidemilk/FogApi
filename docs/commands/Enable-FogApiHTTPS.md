# Enable-FogApiHTTPS

## SYNOPSIS
Enforce https in the url used in all api calls

## SYNTAX

```
Enable-FogApiHTTPS [-ProgressAction <ActionPreference>] [<CommonParameters>]
```

## DESCRIPTION
Prepends https to the fogserver property of fog server settings
